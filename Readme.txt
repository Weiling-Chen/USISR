USISR Dataset：
\Images: Sonar images generated with six typical super-resolution algorithms after HEVC compression.
(super-resolution algorithms: Bicubic [1], FSRCNN [2], SRFBN [3], SwinIR [4], Real-ESRGAN [5], and SAFMN [6])
(scaling factors: 2, 4, and 8)

\Ref: Reference images without downsampling.

USISR_MOS.xlsx: subjective scores of super-resolution reconstructed sonar images in \Images.
